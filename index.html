精彩博文

深入浅出 Web Assembly 近期 Web Assembly 这门新技术慢慢进入了大家的视野，W3C 也成立了专门的 Webassembly Community Group，被标准认可；也是因为这次各大主流浏览器厂商（难得的）达成了一致，共同参与规范的讨论，在自家的浏览器里都实现了。本文采用了漫画的方式帮助读者了解为什么 Web Assembly 这么快，以及 Web Assembly 的底层原理。
使用 React VR 搭建 3D 画廊 React VR 是 Facebook 开发的开发 WebVR 的 JavaScript 库，旨在减少开发成本提高开发效率，本文介绍了用 React VR 搭建 WebVR 应用的全过程。
将饿了么应用升级为 PWA 饿了么在 Vue.js 的积极尝试之后，又开始致力于将饿了么的移动 web 升级为 PWA，本文介绍了饿了么在实践 PWA 的过程中的一写经验和技巧，帮助读者更高效地开发 PWA。 用 React 编写一个 PWA 大神 Addy Osmani 编写的 PWA 开发教程，共分为四个章节，由浅入深，直中要害，值得一读。 JavaScript 启动性能优化 性能一直是前端开发者老生常谈的一个问题，本文分解了 V8 的工作流程，从语言解析、编译的过程着手探讨了如何优化 JavaScript 的性能。 三种减少 webpack bundle 大小的方法 bundle 的大小对页面性能的影响是非常大的，如何在 webpack 构建阶段尽量减小 bundle 大小？ 使用 Fetch API 目前各大浏览器已经对 Fetch API 有了良好的支持，本文详细介绍了 Fetch API 的使用方式，包括基本的请求数据、错误处理等。
